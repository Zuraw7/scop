\doxysubsection{Detailed Description}
Manages loading and binding of textures. 

\doxylink{classTextureManager}{Texture\+Manager} handles loading textures from files, storing them, and binding them to GPU texture slots. It ensures that the same texture is not loaded multiple times and keeps track of assigned slots.

Loads a 2D texture or returns an existing one if already loaded. 

This function checks if a texture with the given path is already loaded in the manager. If it exists, the existing texture is returned. Otherwise, it attempts to create a new \doxylink{classTexture2D}{Texture2D} using {\ttfamily \doxylink{classTexture2D_a53873d77797966d77a5caa1eaa1f448a}{Texture2\+D\+::create()}}. If creation succeeds, the texture is stored in the manager, assigned a slot, and returned. If creation fails (e.\+g., file not found or invalid format), {\ttfamily nullptr} is returned and an error message is printed.

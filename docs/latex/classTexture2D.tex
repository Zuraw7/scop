\doxysubsection{Detailed Description}
Wraps a 2D texture in Open\+GL. 

The \doxylink{classTexture2D}{Texture2D} class loads an image from a file and creates an Open\+GL texture. It supports setting wrapping and filtering options.

Loads an image from a file and creates a 2D Open\+GL texture.

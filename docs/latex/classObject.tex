\doxysubsection{Detailed Description}
Represents a 3D object loaded from an .obj file. 

\doxylink{classObject}{Object} contains vertex and index data, transformation matrices, a texture, and Open\+GL buffers (VAO, VBO, IBO).

Creates and initializes an \doxylink{classObject}{Object} from a .obj file. 

This static function performs the following steps\+:
\begin{DoxyEnumerate}
\item Allocates a new \doxylink{classObject}{Object} instance on the heap.
\item Parses the specified .obj file using {\ttfamily parse\+File()}. Returns {\ttfamily nullptr} if parsing fails.
\item Initializes Open\+GL buffers (VAO/\+VBO/\+IBO) via {\ttfamily init\+Buffers()}.
\item Calculates the geometric center of the object and its scale factor.
\item Sets the object\textquotesingle{}s transformation matrices ({\ttfamily m\+\_\+matrix}, {\ttfamily m\+\_\+translation\+Matrix}, {\ttfamily m\+\_\+rotation\+Matrix}) to identity.
\end{DoxyEnumerate}

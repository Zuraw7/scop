\doxysubsection{Detailed Description}
Wraps an Open\+GL \doxylink{classIndexBuffer}{Index\+Buffer} (element array buffer). 

The \doxylink{classIndexBuffer}{Index\+Buffer} class is responsible for creating, binding, and unbinding an element array buffer in GPU memory, storing the indices used for indexed rendering.

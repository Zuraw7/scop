\doxysubsection{Detailed Description}
Wraps an Open\+GL \doxylink{structVertex}{Vertex} Buffer \doxylink{classObject}{Object} (VBO). 

The \doxylink{classVertexBuffer}{Vertex\+Buffer} class is responsible for creating, binding, and unbinding a VBO in GPU memory. It stores vertex data for rendering and supports construction from raw arrays, std\+::vector of floats, or std\+::vector of \doxylink{structVertex}{Vertex} structs.

Detailed Description

Wraps an OpenGL Vertex Array Object (VAO).

The VertexArray class is responsible for creating, binding, and unbinding a VAO in GPU memory. It stores the state needed to supply vertex data to the GPU, including vertex buffers and attribute pointers.

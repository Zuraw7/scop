Detailed Description

Wraps a 2D texture in OpenGL.

The Texture2D class loads an image from a file and creates an OpenGL texture. It supports setting wrapping and filtering options.

Loads an image from a file and creates a 2D OpenGL texture.

This static function creates a new Texture2D object, loads the image from the specified file using stb_image, uploads it to the GPU, sets wrapping and filtering options, and generates mipmaps. If the image fails to load, the function returns nullptr.

Parameters
pathPath to the texture image file.
wrapSWrapping mode for the S (X) coordinate. Default: GL_MIRRORED_REPEAT.
wrapTWrapping mode for the T (Y) coordinate. Default: GL_MIRRORED_REPEAT.
minFilterMinification filter. Default: GL_LINEAR_MIPMAP_LINEAR.
magFilterMagnification filter. Default: GL_LINEAR.

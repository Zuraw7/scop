Detailed Description

Wraps an OpenGL Vertex Buffer Object (VBO).

The VertexBuffer class is responsible for creating, binding, and unbinding a VBO in GPU memory. It stores vertex data for rendering and supports construction from raw arrays, std::vector of floats, or std::vector of Vertex structs.

Detailed Description

Wrapper for ImGui to simplify GUI creation.

This class provides an easy interface for initializing ImGui, creating frames, displaying text and FPS, and rendering the GUI.

Constructs the cImGUI object with default settings.

Sets initial GUI scale to 0 and initializes default window flags:

  • No decoration
  • Auto-resize
  • No background
  • No saved settings

◆ init()

void cImGUI::init ( GLFWwindow *  window)

Initializes ImGui for the specified GLFW window.

Performs the following steps:

  1. Creates the ImGui context.
  2. Applies DPI scaling.
  3. Sets the GUI style (dark theme by default).
  4. Initializes the GLFW + OpenGL3 backend for rendering.
Parameters
windowPointer to the GLFW window to attach ImGui to.
